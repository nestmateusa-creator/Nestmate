How to Use Demo Dashboards

  1. Click any "View Demo Dashboard" button above to explore that subscription tier
  2. No login required - you'll be taken directly to the interactive demo dashboard
  3. Each dashboard showcases different features based on the subscription tier

  5. All features are fully functional in demo mode
  6. Note: Data is stored in sessionStorage and will be cleared when you close the browser tab

Demo Mode Information

These demo dashboards are fully functional but operate in demo mode. All data is stored temporarily in your browser's sessionStorage and will be cleared when you close the tab. No data is saved to our servers.
