Troubleshooting

If you're still having issues, check the browser console (F12) for error messages and try these steps:

  • Make sure Firebase is properly configured in scripts/firebaseconfig.js
  • Check that Authentication and Firestore are enabled in your Firebase Console
  • Verify your Firebase project ID matches the configuration
  • Check if there are any CORS or security rule issues
